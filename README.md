# Aprendizaje-automatico---Film-Junky-Union
Sentiment analysis project using NLP and machine learning to classify IMDB movie reviews as positive or negative. Includes TF-IDF, Logistic Regression, Gradient Boosting and evaluation with F1, ROC AUC and PRC metrics.
Machine learning project focused on sentiment analysis of movie reviews. 
The project applies NLP techniques such as text normalization, lemmatization and TF-IDF vectorization, and compares multiple classifiers (Logistic Regression, LightGBM) to automatically detect negative and positive reviews with high accuracy. 
Developed as part of a Data Science certification project.
